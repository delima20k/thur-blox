# Store Privacy

## Dados coletados

Obrigatorios: nome do cliente e Roblox username.

Opcionais: Roblox display name, e-mail e telefone.

## Uso

O Roblox username e usado somente para entrega manual dentro do jogo. O nome do cliente nao deve aparecer em telas publicas.

## Pendencias antes de producao

- Politica de privacidade.
- Prazo de retencao.
- Procedimento de exclusao quando aplicavel.
- Termos de uso com ausencia de vinculo oficial com Roblox.
